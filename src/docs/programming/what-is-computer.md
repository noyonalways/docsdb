- A machine that computes something
- Task:
  - Accept data/input
  - Process data
  - Produce output/result
  - Store result
- Usage:
  - Writing Documents
  - Browser the internet

## Components of a Computer

- Hardware
  - CPU
    - Central Processing Unit
    - Control Unit
    - Arithmetic Logic Unit (ALU)
  - Memory
    - Random Access Memory (RAM)
    - Read-Only Memory (ROM)
  - Storage
    - Hard Disk Drive (HDD)
    - Solid State Drive (SSD)
  - Input/Output devices
  - Peripheral devices
    - Mouse
    - Keyboard
    - Monitor
    - Printer
    - Scanner
    - Webcam
- Software
  - System Software
    - Device Drivers
    - Utilities
  - Operating System
    - Windows
    - macOS
    - Linux
  - Applications
    - Word Processor
    - Spreadsheet
    - Presentation Software
    - Graphics Software
    - Web Browser
    - Media Player
